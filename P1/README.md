## Uso del makefile
Hay 8 archivos de test, para ejecutar el tercero, por ejemplo, basta con usar los siguientes comandos:
```
  make test_03
  ./test
```
Para limpiar la carpeta del proyecto de todos los archivos generados basta con usar:
```
  make clean
```
